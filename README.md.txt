# SnappFood

SnappFood is a simple rest api to Create, Read,and Update products and orders written in C#. In this project, I strived to use onion architecture.
.
## Cloning

To clone the source code on your computer in order to build and test it, you can use `git` command-line interface.

```bash
git clone https://github.com/Mona63/SnappFood.git
```

## Building

Before building SnappFood, you need to have DotNet 6.0 installed on your computer.  However, if you are a bash nerd here is an example.

```bash
cd SnappFood
dotnet build
```

## Testing

I've written some Unit tests for this project. You can use your IDE or `dotnet` CLI to run them, here is an example that uses the command line.

```bash
dotnet test
```
